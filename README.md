# 🚀 First Portfolio

The goal of this project will be to create a website that will showcase my skills and learn some basic skills for my first project.
The inspiration for this project is to create an interactive, Windows-like desktop that makes playing with the portfolio engaging.
The features I want to have are as follows:
  - Interactive UI, opening windows as Windows does
  - About Me
  - Links
  - Experience
  - Projects
